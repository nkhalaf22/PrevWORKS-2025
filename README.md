# PrevWorks

PrevWORKS is an application to track the mental health of medical students. This is useful for tracking burnout over a large cohort

Global Steps:

1. Clone GitHub repo on your local machine
2. Install npm

How to run locally:

3. Use "npm run dev"
4. Navigate to localhost:5173

How to run on website

3. Use "npm run build"
4. Use "firebase deploy --only hosting"
5. Navigate to https://prevworktest.web.app/
